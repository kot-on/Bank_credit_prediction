# 💳 Credit Scoring & Clustering Analysis
Данная работа демострирует пример машинного обучения для автоматизации выдачи кредитов клиентам вымышленного банка исходя из них данных.

### Прицпин работы
Данная работа начинается созданием датасета на 150 записей, где каждый пользователь имеет следующие данные: 
- Имя
- доход
- долговая нагрузка (Процент заработанных денег уходящий на закрытие долгов)
- Семейное положение
- наличие детей
- стоимость имущества
- наличие дома
- наличие машины
- возраст
- сумма кредита
- срок кредита
- ставка кредита
- наличие поручителя

После чего идет расчет, а сможет ли клиент платить ежемесячный платеж и выводится статистика скольким людям из датасета можно выдать кредит.
После чего происходит начинается процесс кластеризации и обучение дерева принятия решения и вычисление энтропии Шенона для набора данных и кластеров. Далее происходит создание поверхности, где визуализируют выдачу кредитов.

---
### 🧰 Инструменты

В данной работе использовался язык программирование Python. Для выполнения также были использованы следующие библиотеки: Pandas, numpy, Matplotlib, scikit-learn, scipy.
